Relevant Course Work

  • CSCI 1110
    • The course offers a comprehensive introduction to computer science, covering both the hardware and software aspects of computers.
    • It emphasizes programming skills, application of these skills in problem-solving, and the study of algorithmic concepts and linear data structures.
    • Tools: Java
  • CSCI 1170
    • The course teaches students essential web concepts and skills necessary for designing, implementing, and maintaining web content, including UI and UX best practices, HTML, style sheets, and client-side programming.
    • It covers multimedia foundations, dynamic content, and the organization and maintenance of web content.
    • Tools: HTML, CSS, JavaScript
  • CSCI 1120
    • The course covers fundamental concepts in computer systems, focusing on how the hardware and software environment influences program behavior, and includes data and program representation, program translation, and operating systems.
    • Additional topics include software engineering and networking, providing a broad understanding of key aspects in computer systems.
    • Tools: C, Git
